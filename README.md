## Requirements 

* **Orange Pi Zero 2W**

* **Ubuntu 24.04 LTS arm64**

## How to Use 

```bash
cd ~
git clone https://github.com/CouDeGras/Thymoeidolon.git
cd Thymoeidolon
sudo bash install.sh
```

* This will install and configure the Thymoeidolon service.
